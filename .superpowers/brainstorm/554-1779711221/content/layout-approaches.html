Layout Approach — Single-Screen Master View

All ~90 parameters visible at once. LFO routing panel on right. Pick the layout structure that feels right.

A
Horizontal Bands

Each synthesis section is a labeled horizontal band across the full width. Parameters flow left→right within each band. LFO routing panel pinned to the right edge. Closest to a high-density plugin like U-He Zebra or TAL-NoiseMaker.
